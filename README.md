## 🍎 Feira Fácil Online - Sistema de Gestão para Feirantes e Produtores

O **Feira Fácil Online** é uma plataforma completa e intuitiva projetada para transformar a gestão de pequenos produtores e feirantes. O sistema integra desde o controle de insumos e produção até a venda final em feiras e eventos, oferecendo uma visão clara da saúde financeira do negócio.

---

## 🚀 Funcionalidades Principais

### 1. 📦 Gestão de Estoque Inteligente
*   **Insumos e Matérias-Primas**: Controle rigoroso de ingredientes com unidades de medida flexíveis (Kg, G, L, mL, Un).
*   **Alertas de Estoque Baixo**: Indicadores visuais que avisam quando um ingrediente está acabando, evitando paradas na produção.
*   **Histórico de Compras**: Registro de entradas de insumos com atualização automática do custo médio por unidade.
*   **Produtos Finais**: Catálogo de produtos prontos para venda com suporte a múltiplas tabelas de preço.

### 2. 👨‍🍳 Módulo de Produção e Receitas
*   **Fichas Técnicas (Receitas)**: Criação de receitas detalhadas vinculando insumos aos produtos finais.
*   **Cálculo de Custo Automático**: O sistema calcula o custo de produção de cada produto com base no preço atual dos ingredientes.
*   **Metas de Produção**: Definição de objetivos de produção com acompanhamento de progresso em tempo real.
*   **Baixa Automática**: Ao finalizar uma produção, o sistema retira automaticamente os ingredientes do estoque e adiciona os produtos prontos.

### 3. 🛒 Lista de Mercado Consolidada (NOVO!)
*   **Carrinho de Produção**: Adicione múltiplas receitas à sua lista de compras de uma só vez.
*   **Soma Inteligente**: O sistema consolida todos os ingredientes necessários, somando as quantidades e convertendo unidades (ex: 500g + 0.7kg = 1.2kg).
*   **Exportação para WhatsApp**: Gere uma única mensagem organizada com todos os itens faltantes para levar ao mercado.

### 4. 🎪 Gestão de Eventos e Feiras (PDV)
*   **Eventos Multidias**: Suporte para feiras que duram vários dias, com fechamento de caixa diário.
*   **Tabelas de Preço Configuráveis**: Escolha entre preços **Local**, **Regional** ou **Revenda** no momento da criação do evento.
*   **Controle de Carga**: Registre exatamente o que foi levado para a feira e o que retornou (sobra).
*   **Vendas em Tempo Real**: Interface de vendas rápida com suporte a Dinheiro, Pix e Cartão.

### 5. 💰 Gestão Financeira Centralizada
*   **Fluxo de Caixa**: Registro automático de todas as vendas de eventos e compras de insumos.
*   **Entradas Avulsas com Baixa de Estoque**: Registre vendas fora de feiras selecionando produtos e quantidades; o sistema cuida da baixa no estoque e do cálculo do valor.
*   **Categorização de Gastos**: Organize suas despesas (Logística, Ajudantes, Taxas) para entender para onde vai seu dinheiro.

## 🆕 Novidades da Última Atualização

*   **Lista de Mercado Inteligente**: Agora você pode planejar compras para várias receitas ao mesmo tempo, com soma automática de ingredientes.
*   **Nova Central de Ajuda**: Guia detalhado de cada botão e aba do sistema para facilitar o aprendizado.
*   **Configuração de Preço Antecipada**: Agora você define se a feira é Local, Regional ou Revenda logo na criação, simplificando a tela de vendas.
*   **Vendas Manuais Inteligentes**: O menu de Finanças agora permite selecionar produtos específicos em "Nova Entrada", garantindo que o estoque esteja sempre certo.

## 🛠️ Tecnologias Utilizadas
*   **Frontend**: React + TypeScript + Tailwind CSS
*   **Interface**: Componentes Shadcn/UI para uma experiência moderna e responsiva.
*   **Backend/Banco de Dados**: Supabase (PostgreSQL) com persistência em tempo real.
*   **Ícones**: Lucide React.

## 📈 Como Começar
1.  **Cadastre seus Insumos**: Comece alimentando o estoque de ingredientes.
2.  **Crie suas Receitas**: Vincule os insumos aos seus produtos.
3.  **Planeje suas Compras**: Use a nova **Lista de Mercado** para saber exatamente o que comprar.
4.  **Produza**: Use o módulo de produção para gerar seu estoque de vendas.
5.  **Abra um Evento**: Configure sua feira e comece a vender!
